# 购物车系统 - Bug 演示项目

这是一个用于授课的，故意设计包含明显 bug 的购物车系统演示项目，用于展示和测试 bug 检测能力。

## 项目简介

这是一个基于 React + TypeScript + Vite 构建的前端购物车系统，包含完整的商品展示、购物车管理等功能。项目中故意植入了一个**严重影响主要功能的计算错误 bug**。

## 技术栈

- **React 18** - UI 框架
- **TypeScript** - 类型安全
- **Vite** - 构建工具
- **CSS3** - 样式设计

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

项目将在 `http://localhost:5173` 启动（具体端口可能不同，请查看终端输出）。

### 构建生产版本

```bash
npm run build
```

### 预览生产构建

```bash
npm run preview
```

## 开发说明

这个项目专门用于演示和测试 bug 检测能力。项目代码结构清晰，bug 位置明确，适合用于：

- Bug 检测工具测试
- 代码审查练习
- 调试技能训练
- 测试用例编写
